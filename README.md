# Medical Quiz App

Interaktivni kviz iz medicinske radiologije izrađen u Reactu.

## O projektu
- 10 pitanja iz radiologije i medicinske dijagnostike
- Zeleno/crveno feedback na odgovore
- Score praćenje
- Ekran sa rezultatom na kraju

## Tehnologije
- React
- Vite
- CSS
